# 아이콘 추가 방법

이 폴더에 `icon128.png` 파일을 추가해주세요.

## 아이콘 생성 옵션:

1. **온라인 도구 사용**:
   - https://www.favicon-generator.org/
   - https://favicon.io/
   
2. **직접 제작**:
   - 128x128 픽셀 PNG 파일 제작
   - 캘린더 관련 아이콘 사용 권장

3. **임시로 아이콘 없이 테스트**:
   - `manifest.json`에서 `icons` 섹션을 주석 처리하거나 제거
   - 크롬이 기본 아이콘을 사용함

## 참고:
추가 아이콘 크기를 원하시면:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)
